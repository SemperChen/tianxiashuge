# 天下书阁
下载应用：[天下书阁.apk](https://raw.githubusercontent.com/SemperChen/tianxiashuge/master/天下书阁.apk)
#### 这是一款网络小说资源非常全的小说阅读器。本小说库>起点中文网+纵横中文网+17k小说网+百度小说网+掌阅中文网+书旗小说网+逐浪小说+和阅，
是目前小说最全的小说阅读器，除此之外我们还提供小说不同源大大提高小说库存量。欢迎使用
